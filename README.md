# 💻 Winget UI - Quick and Dirty Winget UI for Updates

This UI allow you to update winget application via Simple GUI.

### Enviroment
Written in C# in .NET 6.0 using Visual Studio 2022

### License
MIT License

### Build (CLI)
```cmd
dotnet build WingetUI.csproj
```
